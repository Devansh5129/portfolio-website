# Personal Portfolio Website
A responsive personal portfolio website built using HTML, CSS, and JavaScript.
## Features
* Responsive Design
* Modern UI
* Smooth Scrolling Navigation
* Typed.js Animation(js library for animated content)
* Skills Section
* Projects Showcase
* Contact Form
* Social Media Links

## Technologies Used

* HTML(html5)
* CSS(css3)
* JavaScript
* Typed.js

## Sections

* Home
* About
* Skills
* Projects
* Contact
